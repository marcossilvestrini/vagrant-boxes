## Getting Started

1. Clone this repository
2. cd vagrant-SWITCH_SOME_BOXE_FOLDER
3. Generate and paste file id_rsa.pub in security folder
4. Vagrant validate
5. Vagrant up
6. Vagrant status
7. Vagrant ssh
8. Done
